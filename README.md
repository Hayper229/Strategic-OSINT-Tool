# Strategic-OSINT-Tool
***Создай папку проекта, положи туда app.py, создай внутри папку templates и положи туда index.html.
Если есть фото цели — закинь их в папку media.
Запусти python app.py и перейди на http://127.0.0.1:5000.
Вставь текст в поле и жми кнопку. Комбайн начнет "шуршать": логи в реальном времени покажут, как он читает текст, идет в сеть и сканирует картинки на наличие лиц и госномеров.***
